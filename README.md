# 🎨 CSS Projects Collection

A curated collection of **UI styling projects** created to enhance and practice front-end design and CSS skills.
This repository includes both **manually designed** and **AI-assisted** styling projects — covering a range of modern layouts, themes, and aesthetics.

---

## 📁 Repository Link

🔗 [css_projects](https://github.com/PRATIK-DWIVEDI11/css_projects)

---

## 🧩 Project Categories

### 🧠 Manually Styled Projects (Pure CSS)

These projects were built from scratch using **hardcoded CSS** to strengthen layout and design fundamentals:

* **Blog** – A clean, content-focused layout for blogs.
* **Card** – Simple yet elegant card UI components.
* **CollegeCard** – Responsive college or student profile card design.
* **Netflix Clone** – A UI replica of Netflix using only HTML & CSS.
* **Recipe** – A stylish recipe page layout with food-focused aesthetics.

### 🤖 AI-Generated & Refined Projects

These designs were first generated using **AI design tools**, then refined and implemented manually in **HTML/CSS**:

* **Algodrishya**
* **Gyansetu**
* **Gurudrishti**
* **Cricknivesh11**

Each project reflects different design principles, themes, and layout explorations.
